Node.jses un entorno en tiempo de ejecución multiplataforma para ejecutar código javascript del lado del servidor.

Instalación

En Ubuntu: 

sudo apt-get install nodejs

En Windows:

https://nodejs.org/es/download/

Node Modules

Node Modules

  • Módulos nativos (core)
  • Módulos de terceros
  • Módulos propios

Modúlos nativos

const path = require('path');

path.extname('index.html');
// .html

path.join('/foo', 'bar', 'baz/asdf', 'quux', '..');
// /foo/bar/baz/asdf

Modúlos de terceros

const _ = require('lodash');

_.chunk(['a', 'b', 'c', 'd'], 2);
// [['a', 'b'], ['c', 'd']]

Node Package Manager (NPM)

npm init

Instalación de paquetes

Dependencias productivas

npm install --save <package_name>
npm install -S <package_name>

Dependencias de desarrollo

npm install --save-dev <package_name>
npm install -D <package_name>

Global Package

npm install --global <package_name>
npm install -g <package_name>

Package.json

  • Guarda las dependencias del proyecto
  • Se pueden definir npm scripts

Modúlos propios

// Utils.js
module.exports = {
	formatPrice(number, currency, decimals) {
		return `${currency}  ${number.toFixed(decimals);}`;
	}
}
// index.js
const Utils = require('./Utils');
Utils.formatPrice(233);
// $223.00

ES6

// Utils.js
const Utils = {
	formatPrice(number, currency, decimals) {
		return `${currency}  ${number.toFixed(decimals);}`;
	}
};
export default Utils;
// index.js
import Utils from './Utils';
Utils.formatPrice(233);
// $223.00
